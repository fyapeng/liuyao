# 六爻排盘与规则辅助分析系统（V1）

基于 Next.js + React + TypeScript 的六爻排盘项目骨架。

## 已实现
- 随机起卦与手动起卦（六次三枚硬币）
- 本卦生成、动爻标记（老阴×、老阳○）、变卦生成
- 本卦/变卦名称识别与并排展示（移动端自动换行）
- 每爻详细信息展示
- 占事问题输入与占事分类选择
- 第一版分析面板（分类、主用神、辅助参考、说明）
- 模块化目录与后续扩展占位模块

## 目录
见 `src/data/*`、`src/lib/*`、`src/components/*`。

## 运行
```bash
npm install
npm run dev
```

## 测试
```bash
npm test
```
